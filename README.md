# 🚀HRM Automation Framework

A Hybrid Test Automation Framework developed using Selenium WebDriver, Java, TestNG, Maven, and Page Object Model (POM) design pattern.

This framework automates core HRM functionalities such as Login, Employee Management, and Employee Search with reusable and maintainable automation architecture.

---

# 📌 Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Git & GitHub
